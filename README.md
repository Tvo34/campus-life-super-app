# campus-life-super-app

## Description
This is a responsive campus web application that helps students explore events, dining options, and campus information.

## Features
- Responsive design using Bootstrap
- Interactive features (events, menu, location)
- Weather information using Open-Meteo API

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap
- Open-Meteo API

## How to Run
Open index.html or visit the deployed GitHub Pages link.
